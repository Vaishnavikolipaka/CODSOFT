## 🖥️ Preview

A fully interactive GUI-based Rock Paper Scissors game with:
- Live scoreboard
- Emoji-based arena display
- Round-by-round results
- Game statistics popup
- Score reset feature

---

## 🚀 How to Run

### Requirements
- Python 3.6 or higher
- No external libraries needed — Tkinter is built into Python

### Run the Game

```bash
python rock_paper_scissors.py
```

---

## 🎮 How to Play

1. Launch the game — a window opens automatically
2. Click **🪨 Rock**, **📄 Paper**, or **✂️ Scissors**
3. The computer picks randomly
4. See the result instantly in the arena
5. Scores update automatically each round
6. Click **📊 View Stats** for a full breakdown
7. Click **🔄 Reset Scores** to start fresh

---

## ✅ Features

| Feature | Details |
|---|---|
| GUI Window | Clean dark-themed Tkinter interface |
| User Input | 3 clickable choice buttons |
| Computer Selection | Random choice each round |
| Game Logic | Rock > Scissors, Scissors > Paper, Paper > Rock |
| Arena Display | Emoji-based visual for both choices |
| Result Banner | Win / Lose / Tie shown with color coding |
| Score Tracking | Player wins, computer wins, ties, rounds |
| Stats Popup | Win rate % and overall standing |
| Reset Button | Clear all scores and start fresh |
| Hover Effects | Buttons highlight on mouse hover |
